# Skills

Reusable capabilities for AI agents.

## Naming

- One skill = one directory under `skills/`
- Name: `kebab-case`
- Entry point: `SKILL.md`

## Adding a new skill

```text
skills/<name>/
├── SKILL.md        # Entry point, spec, help text (required)
└── ...             # Support files as needed
```
